Document内のChoreonoid導入マニュアルを読んでChoreonoidをインストールしたあとに，操作マニュアルを参考にしてご利用ください．


各ファイルの説明

・ChoreonoidCRANEControllerRTC
 　Choreonoid上のCRANE+モデルを制御するコンポーネント

・CraneplusRTC
 　CRANE+実機を制御するコンポーネント

・ManipulatorControlSample
　 ChoreonoidCRANEContollerRTCとCraneplusRTC用のサンプルコントローラコンポーネント

・Document
　 各種マニュアルが入ったファイル

・model_project
　 Choreonoid用のモデルファイルやプロジェクトファイルが入ったファイル

・script
　 コレオノイドのインストール，コンポーネントのビルド，接続等が行えるシェルスクリプトが入ったファイル
